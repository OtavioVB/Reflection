# Reflection

O processo de reflexão (Reflection) fornece descrições de assemblies, módulos e tipos a nível de execução. Ele é utilizado para criar instâncias de objetos dinamicamente, associar um tipo a um objeto e obter um tipo de objeto e invocar seus métodos ou também acessar suas propriedades.

## Referências 
`using System;`
`using System.Reflection;`
